# Railora 🚆

Railora is a railway ticket booking Android application inspired by modern travel apps and IRCTC.

## Tech Stack

### Android
- Kotlin
- XML
- MVVM Architecture
- Navigation Component
- ViewBinding
- Hilt (Upcoming)
- Retrofit + OkHttp (Upcoming)
- Room (Upcoming)
- DataStore (Upcoming)

### Backend (Upcoming)
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Docker
